Stardist Cluster

This application allows you to create the configuration files to run stardist on the bwForCluster MLS&WISO.

Two files are generated:

  • a moab configuration file
  • a sh script

On the creation of these files are copied to the node and then submitted to the cluster. The input and output directories must exist, and the input images in tiff format must be located the input directory beforehand. SDS directories can be specified if mounted in the instance.
